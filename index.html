PASOS PARA CREAR UNA CUENTA EN GITHUB

PRIMER PASO

debes ingresar a la siguiente pagina GITHUB


SEGUNDO PASO

debes crear tu cuenta ingresando tus datos


TERCER PASO

en este paso debes eliegir que tipo de cuenta deseas


CUARTO PASO

en este paso debe conestar unas preguntas simples sobre su interes en la cuenta de GITHUB


QUITO PASO

en este paso GITHUB le ofrece si decea ver un tutorial de como crea un reporcitorio


SEXTO PASO

En este paso debes verificar tu cuenta por medio de tu correo electronico

En este caso debes abrir tu correo y habilitar tu cuenta de GITHUB


PASO FINAL

en este paso ya tienes creada tu cuenta de GITHUB
